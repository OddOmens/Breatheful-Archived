# Breatheful - Features & Tips

## Core Features

### Breathing Exercises
- **14 Breathing Modes** with different timing patterns:
  - Quick (2-0-2-0) - Fast breathing for quick sessions
  - Shallow (3-0-3-0) - Gentle breathing pattern
  - Standard (4-0-4-0) - Default balanced breathing
  - Calm (4-4-4-4) - Box breathing with holds
  - Sleep (4-7-8-0) - Designed to help you fall asleep
  - Focus (5-0-5-0) - Longer breaths for concentration
  - Deep Zen (8-4-8-4) - Extended breathing with holds
  - Monk (10-5-8-5) - Advanced deep breathing
  - Alternate Nostril (2-5-2-5) - Short breaths with long holds
  - Calming (7-0-11-0) - Extended exhale for relaxation
  - Box Breathing (4-4-4-4) - Military-style breathing
  - Relaxing (4-7-8) - Similar to sleep mode
  - Energizing (3-0-2-0) - Quick inhale, faster exhale
  - Ocean Breath (5-2-5-2) - Balanced with short holds

### Voice Guidance
- **Breathing Cue Voices** - Guides you through inhale, hold, and exhale:
  - Kai - Calm, soothing masculine voice
  - Zen - Peaceful, meditative masculine voice
  - Luma - Gentle, bright feminine voice
  - Amara - Soft, nurturing feminine voice

- **Timed Guided Sessions** - Complete guided breathing sessions:
  - 1, 2, and 5-minute sessions with Luma
  - 1, 2, and 5-minute sessions with Kai
  - Sessions automatically return to "No Guide" when complete

### Panic Attack Support
- **Swipe left** from the main screen to access panic attack mode
- **5-4-3-2-1 Grounding Exercise**:
  1. Name 5 things you can see
  2. Find 4 things you can touch
  3. Listen for 3 sounds you can hear
  4. Notice 2 things you can smell
  5. Focus on 1 thing you can taste
- **Calming Breath Pattern** (4-2-6-1) - Extended exhale to activate parasympathetic nervous system
- **Swipe right** to return to main breathing screen

## Tips & Tricks

### Getting Started
1. **Start with Standard Mode** - The default 4-4 breathing pattern is perfect for beginners
2. **Use Voice Guidance** - Select a voice guide to help you maintain rhythm
3. **Choose Background Audio** - Pick an ambient sound that helps you relax
4. **Set a Daily Goal** - Start small (5-10 minutes) and build up gradually

### For Better Sleep
1. Use **Sleep Mode** (4-7-8 breathing)
2. Choose calming audio like **Rain**, **Waves**, or **Meditation**
3. Practice 10-15 minutes before bed
4. Turn off inhale/exhale text for less distraction

### For Anxiety & Stress
1. Access **Panic Attack Mode** by swiping left
2. Use the **5-4-3-2-1 Grounding Exercise**
3. Try **Calming Mode** (7-0-11-0) with extended exhales
4. Enable voice guidance for additional support

### For Focus & Energy
1. Use **Focus Mode** (5-0-5-0) or **Energizing Mode** (3-0-2-0)
2. Try shorter sessions (2-5 minutes)
3. Practice before important tasks or meetings

### Customization Tips
1. **Match Your Mood** - Change colors and circle styles to suit your state of mind
2. **Minimize Distractions** - Hide text guides and phrases for a cleaner interface
3. **Auto-Dim Navigation** - Bottom navigation fades after 2 seconds; tap screen to bring it back
4. **Custom Menu** - Tap the "More" button (•••) to access additional settings

### Tracking Progress
1. **Check Your Calendar** - See which days you practiced
2. **Monitor Weekly Average** - Track consistency over time
3. **Sync to Health** - Keep all your wellness data in one place
4. **Set Reminders** - Build a consistent practice with daily notifications

### Advanced Features
1. **Timed Guided Sessions** - Try 1, 2, or 5-minute guided sessions for structured practice
2. **Monk Mode** - Advanced practitioners can try the 10-5-8-5 pattern
3. **Custom Combinations** - Mix different breathing modes with various audio tracks
4. **Year-over-Year Tracking** - Compare your practice across different years

### Best Practices
- **Breathe through your nose** when possible
- **Find a comfortable position** - sitting or lying down
- **Practice regularly** - Even 5 minutes daily is beneficial
- **Don't force it** - Breathing should feel natural and comfortable
- **Use during transitions** - Between tasks, after meals, before sleep
- **Combine with meditation** - Use phrases for mindfulness practice

### Interface Tips
- **Tap anywhere** on the main screen to show/hide navigation
- **Swipe left** for panic attack support
- **Swipe right** to return from panic mode
- **Tap phrases** to see different inspirational quotes
- **Navigation auto-hides** after 2 seconds for distraction-free breathing
