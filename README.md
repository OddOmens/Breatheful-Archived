# Breatheful

Breatheful is a mindfulness and breathing assistant application for iOS, designed to help users find inner peace through guided breathing exercises and soothing aesthetics.

## Status: Archived

**Breatheful is no longer actively maintained or supported.**

This codebase is being released to the public as-is. It serves as an educational resource and an archive of the project. There is no support provided, and no future updates are planned by the original authors.


## Getting Started

To explore or build the app locally:

1.  **Open the Project**: Double-click `Breatheful.xcodeproj` to open it in Xcode.
2.  **Configure Signing**:
    *   Select the project in the Project Navigator.
    *   Select the `Breatheful` target.
    *   Go to the **Signing & Capabilities** tab.
    *   Select your own Development Team.
    *   If necessary, update the Bundle Identifier to something unique.
3.  **Build and Run**: Select a simulator or connected device and press Run (Cmd+R).


## Features

*   **Breathing Modes**: Multiple guided breathing techniques including 4-7-8, Box Breathing, and more.
*   **Visualizations**: Dynamic, calming animations that sync with your breath.
*   **Audio**: Soothing background sounds (Rain, Forest, Piano, etc.).
*   **HealthKit**: Integration with Apple Health to track Mindfulness Minutes.
*   **Customization**: Extensive theming options for colors and app icons.

## Requirements

*   Xcode 15.0+
*   iOS 17.0+
